## 운영 규칙

- 새 작업 주제가 생기면 먼저 이 파일에 주제 인덱스를 추가한다.
- 주제 인덱스 파일은 배경, 목표 구조, 단계별 세부 문서 링크만 유지한다.
- 구현 단위, 수정 대상 파일, 완료 기준은 세부 TODO 파일에 작성한다.
- 구현이 끝난 항목은 세부 TODO 파일에서 체크하고, 이 파일의 상태를 갱신한다.

# LLM 스킬 로드

외부 디렉토리에서 스킬과 룰 파일을 스캔해 스킬 라이브러리 DB에 저장하는 화면이다.

---

## 화면 구성

| 영역 | 설명 |
|------|------|
| 소스 디렉토리 | 스캔할 폴더를 지정한다. |
| DB 라이브러리 | 현재 사용 중인 스킬 라이브러리 저장 위치를 보여준다. |
| 발견된 파일 | 스캔 결과를 트리 형태로 표시한다. |
| 파일 내용 미리보기 | 선택한 파일 내용을 보여준다. |
| 로드 | 선택된 파일을 DB 라이브러리에 저장한다. |

---

## 기본 흐름

1. 스킬/룰 파일이 들어 있는 폴더를 `소스 디렉토리`로 지정한다.
2. `스캔`을 눌러 파일 목록을 만든다.
3. 필요한 파일 또는 디렉토리를 선택한다.
4. `로드`를 눌러 DB 라이브러리에 저장한다.

---

## 트리 선택

- 파일 목록은 폴더 구조를 유지한 트리로 표시된다.
- 폴더를 선택하면 하위 파일도 함께 선택된다.
- `전체 선택`과 `전체 해제`로 한 번에 상태를 바꿀 수 있다.
- 파일을 선택하면 오른쪽 미리보기에서 내용을 확인할 수 있다.

---

## 저장 방식

- 선택한 파일은 `LlmSkillLibraryRepository`를 통해 SQLite 기반 라이브러리에 저장된다.
- 저장된 파일은 설치 탭에서 `로드된 Cursor 라이브러리`로 다시 읽힌다.
- 스캔한 파일 내용은 상대 경로를 유지한 채 DB에 들어간다.

---

## 주의 사항

- 소스 디렉토리를 먼저 스캔해야 한다.
- 선택한 파일이 없으면 로드할 수 없다.
- 바이너리 파일은 미리보기에서 내용 대신 크기 정보가 표시될 수 있다.
- `로드` 후 설치 탭을 열면 방금 저장한 라이브러리가 목록에 반영된다.
